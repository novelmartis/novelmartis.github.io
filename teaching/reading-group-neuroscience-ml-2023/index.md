# Reading Group: Neuroscience × Machine Learning

<p class="muted">
Reading group course (2023) focused on key papers at the neuroscience-machine-learning interface.
</p>

<hr class="hr">

<div class="projlist">

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://www.nature.com/articles/srep27755">Comparison of deep neural networks to spatiotemporal cortical dynamics of human visual object recognition (Cichy et al., 2016)</a>
      </div>
    </div>
    <p class="pdesc">
      One of the first large-scale brain-network mapping studies showing hierarchical correspondences between DNN layers and human visual processing in both space and time.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://www.nature.com/articles/srep27755">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://www.pnas.org/doi/10.1073/pnas.1905544116">Recurrence is required to capture the representational dynamics of the human visual system (Kietzmann et al., 2019)</a>
      </div>
    </div>
    <p class="pdesc">
      Showed that recurrent architectures better capture time-resolved ventral-stream dynamics than purely feedforward models, supporting a central role for recurrence in vision.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://www.pnas.org/doi/10.1073/pnas.1905544116">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://arxiv.org/abs/1611.09430">Emergence of foveal image sampling from learning to attend in visual scenes (Cheung et al., 2017)</a>
      </div>
    </div>
    <p class="pdesc">
      A striking emergence result: a task-optimized attention model learns retinal-like eccentricity-dependent sampling without imposing it by hand.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://arxiv.org/abs/1611.09430">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://www.science.org/doi/10.1126/science.aar6170">Neural scene representation and rendering (Eslami et al., 2018)</a>
      </div>
    </div>
    <p class="pdesc">
      Introduced the Generative Query Network, showing how agents can infer compact latent scene representations and render novel viewpoints.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://www.science.org/doi/10.1126/science.aar6170">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://www.pnas.org/doi/10.1073/pnas.1719397115">Recurrent computations for visual pattern completion (Tang et al., 2018)</a>
      </div>
    </div>
    <p class="pdesc">
      Linked delayed recognition under occlusion and masking effects to recurrent computations; recurrent models captured behavior and neural timing better than feedforward ones.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://www.pnas.org/doi/10.1073/pnas.1719397115">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://arxiv.org/abs/2108.08810">Do vision transformers see like convolutional neural networks? (Raghu et al., 2021)</a>
      </div>
    </div>
    <p class="pdesc">
      Compared representation geometry in ViTs and CNNs, highlighting earlier global integration in ViTs and stronger layer-wise uniformity.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://arxiv.org/abs/2108.08810">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://arxiv.org/abs/1706.01427">A simple neural network module for relational reasoning (Santoro et al., 2017)</a>
      </div>
    </div>
    <p class="pdesc">
      Showed that explicit relation modules can dramatically improve relational reasoning capabilities in otherwise strong convolutional architectures.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://arxiv.org/abs/1706.01427">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://www.pnas.org/doi/10.1073/pnas.1803839115">Alleviating catastrophic forgetting using context-dependent gating and synaptic stabilization (Masse et al., 2018)</a>
      </div>
    </div>
    <p class="pdesc">
      Combined synaptic stabilization with context-dependent sparse gating to reduce interference across tasks, offering an accessible continual-learning recipe.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://www.pnas.org/doi/10.1073/pnas.1803839115">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://www.pnas.org/doi/10.1073/pnas.2201968119">A hierarchy of linguistic predictions during natural language comprehension (Heilbron et al., 2022)</a>
      </div>
    </div>
    <p class="pdesc">
      Used GPT-2-derived prediction measures with neural data to show that language comprehension involves concurrent multi-level predictive processing.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://www.pnas.org/doi/10.1073/pnas.2201968119">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://www.nature.com/articles/s41586-018-0102-6">Vector-based navigation using grid-like representations in artificial agents (Banino et al., 2018)</a>
      </div>
    </div>
    <p class="pdesc">
      Demonstrated that goal-directed agents can develop grid-like codes that support vector navigation and shortcuts, linking function to entorhinal-style representations.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://www.nature.com/articles/s41586-018-0102-6">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://papers.nips.cc/paper/6849-toward-goal-driven-neural-network-models-for-the-rodent-whisker-trigeminal-system">Toward goal-driven neural network models for the rodent whisker-trigeminal system (Zhuang et al., 2017)</a>
      </div>
    </div>
    <p class="pdesc">
      Extended goal-driven modeling beyond primate vision to rodent somatosensation, broadening the NeuroAI template across species and sensory systems.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://papers.nips.cc/paper/6849-toward-goal-driven-neural-network-models-for-the-rodent-whisker-trigeminal-system">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://www.cell.com/neuron/fulltext/S0896-6273(21)00501-8">Single cortical neurons as deep artificial neural networks (Beniaguev et al., 2021)</a>
      </div>
    </div>
    <p class="pdesc">
      Showed that single-neuron input-output mappings can require deep temporal architectures, emphasizing the computational richness of biophysical neurons.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://www.cell.com/neuron/fulltext/S0896-6273(21)00501-8">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://journals.plos.org/ploscompbiol/article?id=10.1371/journal.pcbi.1005268">Could a neuroscientist understand a microprocessor? (Jonas and Kording, 2017)</a>
      </div>
    </div>
    <p class="pdesc">
      A methodological stress test for neuroscience analyses, showing that standard tools can recover structure yet still miss true mechanistic understanding.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://journals.plos.org/ploscompbiol/article?id=10.1371/journal.pcbi.1005268">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://www.science.org/doi/10.1126/sciadv.abm2219">Reassessing hierarchical correspondences between brain and deep networks through direct interface (Sexton et al., 2022)</a>
      </div>
    </div>
    <p class="pdesc">
      Revisited hierarchical brain-network correspondence using a direct interfacing framework, showing richer cross-stage category information than strict feedforward mappings suggest.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://www.science.org/doi/10.1126/sciadv.abm2219">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://link.springer.com/article/10.1007/BF00114844">Distributed representations, simple recurrent networks, and grammatical structure (Elman, 1991)</a>
      </div>
    </div>
    <p class="pdesc">
      Classic psycholinguistics paper showing how recurrent dynamics can capture grammatical regularities from sequential input.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://link.springer.com/article/10.1007/BF00114844">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://www.pnas.org/doi/10.1073/pnas.1800755115">Comparing continual task learning in minds and machines (Flesch et al., 2018)</a>
      </div>
    </div>
    <p class="pdesc">
      Directly compared human and machine multi-task learning, highlighting factorized representations as a key ingredient for reduced interference.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://www.pnas.org/doi/10.1073/pnas.1800755115">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://www.nature.com/articles/ncomms13276">Random synaptic feedback weights support error backpropagation for deep learning (Lillicrap et al., 2016)</a>
      </div>
    </div>
    <p class="pdesc">
      Introduced feedback alignment, showing useful credit assignment can emerge without exact weight symmetry required by standard backpropagation.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://www.nature.com/articles/ncomms13276">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://www.science.org/doi/10.1126/science.aav9436">Neural population control via deep image synthesis (Bashivan et al., 2019)</a>
      </div>
    </div>
    <p class="pdesc">
      Showed that deep image synthesis can drive targeted V4 neural populations beyond natural response levels, revealing highly specific and sometimes counterintuitive features.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://www.science.org/doi/10.1126/science.aav9436">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://www.nature.com/articles/s41598-023-42891-8">Brain-Diffuser: Natural scene reconstruction from fMRI signals using generative latent diffusion (Ozcelik et al., 2023)</a>
      </div>
    </div>
    <p class="pdesc">
      Used diffusion models with CLIP-aligned representations to reconstruct viewed images from fMRI, including ROI-specific reconstructions of hierarchical feature selectivity.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://www.nature.com/articles/s41598-023-42891-8">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://arxiv.org/abs/2109.11523">How much human-like visual experience do current self-supervised learning algorithms need in order to achieve human-level object recognition? (Orhan, 2022)</a>
      </div>
    </div>
    <p class="pdesc">
      Estimated the data-efficiency gap between human visual learning and current SSL pipelines, arguing that modern systems remain far more data hungry.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://arxiv.org/abs/2109.11523">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://www.science.org/doi/10.1126/science.aab0551">Cortical information flow during flexible sensorimotor decisions (Siegel et al., 2015)</a>
      </div>
    </div>
    <p class="pdesc">
      A landmark multi-area recording paper showing interacting bottom-up sensory and top-down task-information flows underlying flexible decisions.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://www.science.org/doi/10.1126/science.aab0551">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://openreview.net/forum?id=r1iuQjxCZ">On the importance of single directions for generalization (Morcos et al., 2018)</a>
      </div>
    </div>
    <p class="pdesc">
      Reported that robustly generalizing models are less dependent on single highly selective directions, informing both theory and regularization practice.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://openreview.net/forum?id=r1iuQjxCZ">Paper</a>
    </div>
  </div>

  <div class="pcard compact">
    <div class="phead">
      <div class="ptitle">
        <a href="https://www.sciencedirect.com/science/article/pii/S0896627320308990">Using deep reinforcement learning to reveal how the brain encodes abstract state-space representations in high-dimensional environments (Cross et al., 2021)</a>
      </div>
    </div>
    <p class="pdesc">
      Linked DQN representations to fMRI activity and behavior during Atari play, connecting RL state abstractions with distributed sensorimotor brain dynamics.
    </p>
    <div class="plinks">
      <a class="btn primary" href="https://www.sciencedirect.com/science/article/pii/S0896627320308990">Paper</a>
    </div>
  </div>
