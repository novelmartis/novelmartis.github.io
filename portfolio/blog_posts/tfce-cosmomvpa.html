Threshold-free cluster enhancement (TFCE) in CoSMoMVPA

Technical Note / 2020

A practical note on what TFCE is, why it is useful, and how CoSMoMVPA actually implements it.

Assumes some prior familiarity with TFCE and CoSMoMVPA.

Threshold-free cluster enhancement is a way to perform voxel-level inference while taking local support into account. Instead of deciding first on a hard cluster-forming threshold and then asking whether clusters survive correction, TFCE accumulates evidence continuously from both signal height and spatial extent.

The Principle

The underlying idea is simple: a voxel should count as more convincing when it sits inside a larger and stronger “hill” of neighboring voxels. TFCE therefore rewards both intensity and extent. In practice, this means a voxel’s score depends not just on its own value, but also on how much local support surrounds it.

How CoSMoMVPA Computes It

CoSMoMVPA computes TFCE iteratively over a set of height markers between baseline and the maximum observed value. At each marker, it finds suprathreshold voxels, identifies contiguous clusters, and increments each voxel in a cluster by an amount that depends on the cluster size and the current height.

That process is repeated over markers, effectively integrating support over the entire range rather than relying on a single chosen threshold. The same logic is also applied to negative deviations after flipping values around the baseline.

Null Distribution and Inference

For inference, CoSMoMVPA builds a null distribution using sign-flips across participants after subtracting the baseline. For each iteration it recomputes TFCE values, separately for positive and negative effects, producing null distributions for comparison against the observed map.

The final output is converted into voxel-wise z-scores. Those z-scores can then be thresholded in the usual way, for example at z ≥ 1.96 and z ≤ -1.96 for a two-sided 5% criterion.

Minimal Example

% Let ds.samples contain voxel values across aligned brains.

cluster_nbrhood = cosmo_cluster_neighborhood(ds);

opt = struct();
opt.niter = 10000;
opt.h0_mean = 0;
opt.null = [];

tfce_zscores = cosmo_montecarlo_cluster_stat(ds, cluster_nbrhood, opt);

p = 0.05;
z_th = norminv(1-p/2);

pos_voxs = tfce_zscores.samples >= z_th;
neg_voxs = tfce_zscores.samples <= -z_th;
